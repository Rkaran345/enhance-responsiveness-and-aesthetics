# Aura - Responsive Productivity Workspace

Aura is a modern, responsive productivity web application designed to help users structure their daily tasks, manage deep focus sessions, and customize their acoustic workspace.

## Features

- **Dashboard**: High-level overview of focus statistics, tasks, and an embedded Focus Flow timer.
- **Tasks Manager**: A complete task list where you can prioritize (Low/Medium/High) and categorize (e.g. Design, Development) tasks.
- **Focus Flow**: A dedicated Pomodoro-style timer to track deep work sessions and breaks.
- **Ambience Space**: Customize your focal environment by playing and mixing ambient sounds (Rainfall, Forest Wind, Café Bustle) with individual volume controls.
- **Ambient Glow**: Personalize your workspace aesthetic with dynamic ambient background glows (Purple, Emerald, Rose).

## Tech Stack

- **React** (v19)
- **TypeScript**
- **Vite** (for fast development and build)
- **Tailwind CSS** (for styling and layouts)
- **Lucide React** (for icons)

## Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Rkaran345/enhance-responsiveness-and-aesthetics.git
   cd enhance-responsiveness-and-aesthetics
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build the application for production:
   ```bash
   npm run build
   ```
